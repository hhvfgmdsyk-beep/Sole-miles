⚡ Auto-Log via Shortcut
Your iPhone Shortcut reads today's walking miles from Apple Health, shows you a shoe menu, then opens this page with the URL ?shoe=NAME&miles=X — which auto-logs instantly with a one-tap confirm. Nothing to type.
Build the Shortcut
1
Save this page to Home Screen
In Safari → Share → Add to Home Screen → name it Sole Miles. Copy the URL from the address bar.
2
New Shortcut → "Log My Shoes"
Shortcuts app → + → rename to Log My Shoes.
3
Action 1 — Fetch today's distance from Health
Add Action → Find Health Samples
· Type: Walking + Running Distance
· Filter: Date is today
· Sort: Date, newest first
4
Action 2 — Sum the samples
Add Action → Calculate Statistics
· Input: Health Samples (magic variable) · Statistic: Sum
5
Action 3 — Round to 2 decimal places
Add Action → Round Number
· Number: Calculated Result · Mode: Normal · Decimals: 2
6
Action 4 — Choose from Menu (pick your shoe)
Add Action → Choose from Menu
· Prompt: Which shoes today? (Rounded Result mi)
· Add one item per shoe below — names must match exactly.
7
Action 5 — Open URL inside each menu branch
Shortcuts creates a branch for each shoe. Inside each one, add Open URLs.

Type your base URL, then ?shoe=SHOE+NAME&miles=, then tap the variable picker {x} and insert Rounded Result for the miles value.

The ready-made URLs for each of your shoes are above in Step 6 — just replace ROUNDED_RESULT with the variable.
YOUR_URL?shoe=SHOE+NAME&miles=ROUNDED_RESULT
8
Set the daily notification
Shortcuts → Automation tab → + → Time of Day
· Set time (e.g. 8 PM, daily)
· Add action: Show Notification → Log your shoes today! 👟
· Add action: Run Shortcut → Log My Shoes
· Enable Run Immediately
✓
Daily flow once set up
Notification fires → tap → Shortcut reads Health miles → you pick shoe → Sole Miles opens → one-tap confirm logs everything instantly.
